## Development Workflow

1. **Requirements:** Captured in Salesforce Cases
2. **Design:** Documented in specs (RevX/Spec Ops assists)
3. **Development:** Feature branches in `sanguinebio/salesforce` repo
4. **Review:** GitHub PR with peer review
5. **QA:** Deploy to QA sandbox, validate
6. **Deploy:** Production deployment with approval
